### Description

------------
This program allows us to print a type of data in a given string.

------------
### How to use
**Compile & Execute**

	$ gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c
	 ./a.out

------------
### Supported Types

------------
| Type   | Output |
|--------|--------|
| d or i | Signed decimal integer |
| c      | Single character |
| s      | String of characters |
